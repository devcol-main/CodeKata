###### Description

정수 `n`이 주어질 때, `n`이하의 짝수를 모두 더한 값을 return 하도록 solution 함수를 작성해주세요.

---

##### 제한사항

0 < `n` ≤ 1000

---

##### 입출력 예

|n|result|
|---|---|
|10|30|
|4|6|

---

##### 입출력 예 설명

입출력 예 #1

- `n`이 10이므로 2 + 4 + 6 + 8 + 10 = 30을 return 합니다.

입출력 예 #2

- `n`이 4이므로 2 + 4 = 6을 return 합니다.

---

## 문제 풀이

```cpp

// solution.cpp

#include <string>
#include <vector>

using namespace std;

int solution(int n) 
{
    int answer = 0;      

    //answer = evenNum;

    if (0 < n <= 1000)
    {
        while (n > 0)
        {
            // check even
            int evenNum = n % 2;
            if (evenNum == 0)
            {
                answer += n;

                //printf("solution: %d", answer);
            }

            // break;
            --n;
        }
    }    

    //printf("final: %d", answer);

    return answer;
}


```
