###### Description

정수를 담고 있는 배열 arr의 평균값을 return하는 함수, solution을 완성해보세요.

#### 제한사항

- arr은 길이 1 이상, 100 이하인 배열입니다.
- arr의 원소는 -10,000 이상 10,000 이하인 정수입니다.

#### 입출력 예

|arr|return|
|---|---|
|[1,2,3,4]|2.5|
|[5,5]|5|
---

## 문제 풀이

```cpp

// solution.cpp

#include <string>
#include <vector>

using namespace std;

double solution(vector<int> arr) {
    double answer = 0;
    double sum = 0;
    //iterator 사용 해서 하는 법
    for (std::vector<int>::iterator it = arr.begin(); it != arr.end(); ++it) {
        //std::cout << *it << " ";
        sum += *it;       
    }

    //cout << numbers.size() << endl; // 현재 벡터에 저장된 실제 원소의 개수
    answer = sum / arr.size();
    
    return answer;
}


```
